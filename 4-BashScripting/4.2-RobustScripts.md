# 4.2. Building robust scripts

```bash
nano parentscript
sudo chmod +x parentscript
nano childscript
```

```bash
#!/usr/bin/bash

export ENV_CITY="Oviedo"
echo "i'm the parent script"
source ./childscript
```

```bash
#!usr/bin/bash

town='Salas'
echo "i am the child script, currently at ${ENV_CITY}, precisely at ${town}"
```
